## 1. 프로젝트 한 줄 정의

> 대학생의 과제·시험·일정을 AI가 분석하여 매일 최적의 행동을 추천해주는 **개인 운영 비서 앱**

## 2. v1 → v2 주요 변경 사항

| 항목 | v1 (계획) | v2 (현재 실제) |
|---|---|---|
| **AI 텍스트 분석** | OpenAI GPT-4o | Groq Cloud — llama-3.3-70b-versatile |
| **AI 이미지 파싱** | OpenAI Vision API | Groq Cloud — meta-llama/llama-4-scout-17b-16e-instruct |

| **비용 구조** | $5~15/월 (OpenAI) | $0~2/월 (Groq Free Tier) |
| **개발 진행** | W1~W3 예정 | W1 ✅, W2 ✅, W3 ✅ 완료 |

## 5. 핵심 컨셉 (유지)

### 설계 원칙 3가지

| 원칙 | 설명 |
|---|---|
| **제로 정리** | 사용자가 정리할 필요 없다. 던져넣으면 AI가 구조화한다. |
| **매일 한 문장** | "오늘 2시간은 DB 과제" — 복잡한 대시보드보다 한 줄 추천이 핵심이다. |
| **틀려도 괜찮다** | AI 추천이 100% 정확할 필요 없다. "아, 맞다 그거 해야지"를 떠올리게 하면 성공이다. |

## 6. 성공 기준 (v2 업데이트)

### 기능 검증 (W6 완료 기준)

- [ ] 과제 입력 → 우선순위 산출 → 오늘 추천까지 End-to-End로 동작한다.
- [ ] AI 코멘트 응답이 3초 이내에 돌아온다. (목표 — 실제 측정값 없음)
- [ ] 시험 일정 충돌이 우선순위에 반영된다.
- [ ] 본인이 2주 연속 매일 1회 이상 실제로 사용한다.

### 사용자 검증 (W8 완료 기준)

- [ ] 주변 동기/후배 5명 이상이 베타 테스트에 참여한다.
- [ ] 5명 중 3명 이상이 1주일 후에도 접속한다.
