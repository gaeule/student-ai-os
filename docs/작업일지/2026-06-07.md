## 세션 1

### 완료한 작업

#### 1. `src/lib/studyPlan.ts` — W6 핵심 로직

**신규 타입**
- `FuturePlanBlock`: 날짜별 배치 항목 (assignmentId, title, subjectName, dueDate, allocatedMinutes)
- `FuturePlanItem`: 날짜 그룹 (date, dateStr, daysFromToday, blocks, totalMinutes)
- `UnplacedWarning`: 마감 전 배치 불가 항목 (assignmentId, title, dueDate, daysLeft, unplacedMinutes)
- `DailyPlanResult`에 `futurePlan`, `unplacedWarnings` 필드 추가

**`buildFuturePlan()` 순수 함수**
- overflow 중 과제 블록만 대상 (시험·마감 초과 제외)
- 마감 가까운 순 정렬 → 내일(day+1)부터 마감일까지 분산
- 과제별 하루 최대 60분, 날짜 전체 합산 최대 180분 (`MAX_DAILY_TOTAL_MINUTES`)
- 10분 단위 내림(floor) — 표시값 = 차감량으로 일치
- 남은 시간이 마감 전에 다 배치 안 되면 `unplacedWarnings` 기록
- KST 기준 날짜 문자열 사용 (`KST_FORMAT`)

**`buildDailyPlan` KST 버그 수정**
- `new Date(); setHours(0,0,0,0)` → KST 날짜 문자열 파싱 후 `new Date(ky, km-1, kd)`로 생성
- Vercel(UTC) 환경 오전 0~9시 전날 계산 문제 해결
