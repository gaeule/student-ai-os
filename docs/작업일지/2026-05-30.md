## 버그 수정

### 버그 1 — zod v4 API 변경
- `required_error` → `error`, `invalid_type_error` → `error`로 변경

### 버그 2 — react-day-picker v10 classNames 변경
- `table` → `month_grid` (shadcn calendar.tsx 수정)
- `initialFocus` prop 제거

### 버그 3 — Select 과목 선택 동작 안 함
- **원인**: `value=""` 전달 시 base-ui 내부 `hasSelectedValue = ("" != null) = true` → 선택 없음인데 선택된 상태로 인식
- **해결**: `value={field.value || null}` 전달, `onValueChange={(val) => field.onChange(val)}` 명시적 래핑

### 버그 4 — Popover 달력 열리지 않음
- **원인**: `PopoverTrigger`에 children만 넣는 방식에서 click 이벤트 처리 불안정 (base-ui v1.5.0)
- **해결**: `render` prop에 `<button>` 직접 전달 (base-ui 권장 방식)

### 버그 5 — 난이도 버튼 시각 피드백 없음
- **원인**: `data-[active=true]:클래스명` 을 const 객체 안에 넣으면 Tailwind v4 JIT 스캔 누락
- **해결**: `isActive ? activeClass : inactiveClass` 명시적 조건부 `cn()` 클래스로 교체
