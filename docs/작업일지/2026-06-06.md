## 세션 1

### 완료한 작업

#### 1. `src/lib/studyPlan.ts` 신규 생성 (핵심 로직)
- `AssignmentStudyBlock`, `ExamStudyBlock`, `StudyBlock` (union), `DailyPlanResult`, `OverdueAssignment` 타입 정의
- 시험 복습 버킷(0.5 / 0.8 / 1.5)을 과제 버킷(0~4) 사이에 소수로 끼워 단일 정렬
  - 오늘 과제(0) → 오늘 시험(0.5) → 내일 시험(0.8) → 내일 과제(1) → 준비기간 시험(1.5) → 이후 과제
- `calcExamRequestedMinutes`: 10분 단위 반올림, 최대 90분, 시험 유형 배율(quiz 0.7, practical 1.2)
- `calcExamTimeCap`: 가용 시간 대비 시험 복습 상한(당일 100%, 1일 전 70%, 2일+ 50%)
- `buildDailyPlan`: 마감 초과 분리 → 블록 생성 → 버킷 정렬 → 탐욕 시간 배정 → `{scheduled, overflow, overdue}` 반환
- **버그 수정**: `today = new Date()`에 현재 시각 포함으로 날짜 계산 오류 → `today.setHours(0,0,0,0)` 추가

#### 2. Vitest 설정 및 테스트 작성
- `vitest.config.ts` 신규, `@` 경로 별칭, package.json `"test": "vitest run"` 추가
- `src/lib/studyPlan.test.ts`: 19개 테스트 전원 통과
  - `calcExamRequestedMinutes` 7개, `calcExamTimeCap` 3개, `buildDailyPlan` 9개

#### 3. `src/lib/actions/ai.ts` 수정
- `ScoredAssignment[]` → `StudyBlock[]` 파라미터 변경
- 과제/시험 블록별 포맷 분리 (`[과제]` / `[시험복습]`)

#### 4. `src/app/(app)/today/_components/TodayPlanner.tsx` 수정
- `recommend()` → `buildDailyPlan()` 교체
- `ExamStudyCard` 컴포넌트 추가 (violet 테마, 시험일·범위·복습 시간 표시)
- `formatMinutes()` 헬퍼 추가 (분 단위 → `1시간 30분` 포맷)
- `Summary`, `OverflowPlan` 과제+시험 혼합 처리
