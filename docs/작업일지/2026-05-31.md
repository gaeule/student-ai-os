## 트러블슈팅

| 문제 | 원인 | 해결 |
|---|---|---|
| 회원가입 "invalid" 오류 | Supabase 이메일 발송 한도(3건/시) 초과 | Supabase 대시보드에서 Confirm email OFF |
| 테스트 계정 로그인 실패 | auth.users에 유저 미생성 | Authentication → Add user → Auto Confirm 체크 |
| SQL UPDATE "No rows returned" | 유저 자체가 없어서 대상 없음 | 유저 먼저 생성 후 SQL 또는 Auto Confirm 사용 |
