#### 수정 후 목록 미반영 (상태 동기화 버그)
- **원인**: `useState(initialExams)`는 초기값만 한 번 읽음 → `router.refresh()`로 서버에서 새 `initialExams` 내려와도 `exams` state에 미반영
- **수정**: `useEffect(() => setExams(initialExams), [initialExams])` 추가로 prop 변경 시 state 자동 동기화
- **부수 수정**: `refresh()` 함수에서 `getExams().then(setExams)` 직접 호출 제거 → `router.refresh()` 단독 사용으로 단순화

#### UTC 날짜 하루 오차
- **원인**: `toISOString().split("T")[0]` → KST 자정 = UTC 전날 15:00 → 하루 앞 날짜로 저장
- **수정**: `createExam`, `updateExam`의 `examDate` 타입 `Date → string` 변경, 클라이언트에서 `format(date, "yyyy-MM-dd")` 전달
- **파일**: `exams.ts`, `ExamsClient.tsx`
